## 📌 Project Overview
A JavaFX-based desktop application designed to simulate an Electric Vehicle (EV) charging station system with a clean and interactive user interface.

---

## 🚀 Features
- User Login Interface  
- Multi-page Navigation (Page1–Page10)  
- JavaFX + FXML based UI  
- MVC Architecture (Controller-based logic)  
- Clean and structured code  

---

## 🛠️ Tech Stack
- Java (JDK 17+)
- JavaFX
- FXML
- CSS

---

## 📂 Project Structure
```text
src/
└── Page1/
    ├── Main.java
    ├── Page1View.fxml
    └── Page1ViewController.java
```

---

## ▶️ How to Run

1. Install **JDK 17 or above**
2. Download and setup **JavaFX SDK**
3. Open terminal in project directory
4. Run the application using:

```bash
java --module-path "C:\path\to\javafx\lib" --add-modules javafx.controls,javafx.fxml -cp out Page1.Main
```
